  # MovieFinderAgent

使用 LangChain 构建的电影信息查询 Agent，基于大语言模型（LLM）实现自然语言电影搜索与信息整合。

## 项目简介

这是一个轻量级的智能电影查找助手，用户可以用自然语言提问，例如：

- “最近五年最好看的科幻电影有哪些？”
- “克里斯托弗·诺兰所有电影按评分排序”
- “《盗梦空间》演员阵容和豆瓣评分是多少？”
- “我想看类似《寄生虫》的韩国悬疑片，推荐几部”

Agent 会自动选择合适的工具，调用电影数据库 API 或网页检索，综合返回结构化、准确的电影信息。

## 主要功能

- 自然语言理解与意图识别
- 多工具集成（电影数据库查询、网页搜索、RAG 增强等）
- 支持复杂查询（时间范围、类型、导演、演员、评分排序、相似推荐等）
- 可扩展的工具体系（未来可轻松加入豆瓣、IMDb、TMDB、维基等更多数据源）
